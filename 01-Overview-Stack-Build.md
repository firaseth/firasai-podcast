## 🏗 ️ SECTION 3: 12-Phase Build Guide

### PHASE 1: Foundation (Week 1)

Step 1: Set Up Recording Hardware
Equipment Checklist:
- Shure SM7B microphone ($399)
- Focusrite Scarlett 2i2 interface ($199)
- XLR cable ($15)
- Headphones: Audio-Technica ATH-M50x ($150)
- Acoustic foam panels ($50)
- Microphone stand ($30)
- Pop filter ($15)
Total Hardware Investment: ~$860

Step 2: Create Core Accounts
- ChatGPT Plus
- Claude Pro
- Descript Pro
- Riverside.fm
- Buzzsprout
- Notion
- ElevenLabs
- Beehiiv (or Substack)
- Buffer
- Make.com

Step 3: Build Your Content Hub
Create a Notion workspace with these sections:
- Command Center (dashboard)
- Episodes Database
- Guests Pipeline
- Content Calendar
- Research Database
- Show Bible

### PHASE 2: Pre-Production Pipeline (Week 2)

Step 4: AI Research Workflow
Process:
1. Perplexity AI → gather facts, stats, sources
2. ChatGPT → structure research into outline
3. Notion → save in episode database
4. Tag with topics, format, and priority

Step 5: Script Generation
Use the Scriptwriter Agent prompt to:
- Generate 30-45 minute scripts
- Create multiple hook options
- Extract quotable moments
- Generate show notes

Step 6: Create Signature Assets
- Record your voice samples (30+ minutes)
- Upload to ElevenLabs for cloning
- Generate intro/outro variations
- Save as reusable audio files

### PHASE 3: Production (Week 3)

Step 7: Recording Setup
- Primary: Riverside.fm (4K video + WAV audio, local recording)
- Backup: Zencastr (second cloud copy)
- Format: WAV, 48kHz, 24-bit

Step 8: AI Co-Host Options
Option A: Brain Co-Host
- Keep ChatGPT/Claude open during recording
- Suggest follow-up questions in real-time

Option B: Voice Agent
- Vocode/Retell AI → Deploy an AI agent
- Greets guests, asks pre-set questions
- Joins via API to recording session

Step 9: Edit with Descript
- Auto-transcription on upload
- Delete filler words with one click
- Use "Studio Sound" feature
- Export clean audio

Step 10: Audio Enhancement
- Run through Adobe Podcast Enhance
- Run through Auphonic (target -16 LUFS)
- Optional: Hindenburg for multitrack mastering

### PHASE 4: Post-Production & Distribution (Week 4)

Step 11: Auto-Generate Show Notes with Castmagic
Upload audio → Castmagic produces:
- Episode title
- 3-sentence summary
- Bullet-point show notes
- Timestamped chapters
- Pull-quotes
- Tweet thread
- Blog post
- SEO keywords

Step 12: Transcription & Searchable Archive
- Otter.ai or Whisper → Full transcript
- Upload to Pinecone (vector DB)
- Build RAG system for searchable archive

Step 13: Clip Creation
- Opus Clip → AI finds viral moments, creates vertical clips
- Headliner → Create audiograms for static promo

Step 14: Auto-Publish via Make.com
See Section 6 for complete automation blueprints.
